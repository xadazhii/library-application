##  **Overview**  

**Library Application** is a modern application that allows:  
- **Users** to easily manage their book collections and transactions.  
- **Administrators** to oversee users, books, and borrowing transactions.  

The program provides a user-friendly interface to access personal dashboards, libraries, collections, and more.
